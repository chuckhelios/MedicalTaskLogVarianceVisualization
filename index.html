EHR Data Visualization

The plots show different representations of data for EHR Tasks for the month of June 2012. You could visualize the following information


1. Stacked-Bar Graph (Top). Visualize the variation of tasks over department as well as the spread across time.
2. Multiples (Top). Visualize the relative spread of each Task across the timeline (standard baselines for specific particular Task).
(YOU CAN SWITCH BETWEEN OPTION 1 AND 2 BY CHECKING EITHER OF THE RADIO BUTTONS PROVIDED.)
3. Stream-Graph (Bottom Left). Visualize the variation in Tasks on a continuous spectrum across the timeline. Interactive time plot area to inquire the exact point data values.
4. Box-and-Whiskers Graph (Bottom Right). Visualize the statistical summary of each task, and inquire of the individual totals using interactive dots on data-axes (support for day base and eacounter base).

(USE THE NAVIGATION POD TO SWITCH BETWEEN VIEWS)
